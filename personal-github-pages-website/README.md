# Personal GitHub Pages Website

This is a simple academic/professional website starter for GitHub Pages.

## How to publish it

1. Create a GitHub account or sign in.
2. Create a new public repository named:

   `yourgithubusername.github.io`

3. Upload all files from this folder into that repository.
4. Go to the repository's **Settings**.
5. Click **Pages**.
6. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Save.
8. Your website should appear at:

   `https://yourgithubusername.github.io`

## What to edit first

- Replace `A. P.` with your full name in every HTML file.
- Replace `your.email@example.com` in `contact.html`.
- Replace GitHub, LinkedIn, Google Scholar, and ORCID links.
- Add your profile photo as `assets/profile.jpg`.
- Add your CV as `assets/cv.pdf`.
- Add papers or slides to `assets/papers/`.

## Pages included

- Home
- Research
- Publications
- Projects
- Teaching & Service
- CV
- Contact
